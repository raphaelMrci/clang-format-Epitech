# clang-format-Epitech
Epitech Norm 2022 compliant clang-format config file

Put this file in the root of your project, and enjoy clang-format

before clang-format
```cpp
namespace Abc {
    class Def {
    public:
    void simulate(void);
    };
}
```

after clang-format
```cpp
namespace Abc {
    class Def {
        public:
            void simulate(void);
    };
} // namespace Abc
```
